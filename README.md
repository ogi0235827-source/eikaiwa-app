# AI英会話チューター

AIアバター「エマ先生」と英会話練習ができるスマホ向けWebアプリ(PWA)。

- 💬 自由会話 + シーン別ロールプレイ(空港・ホテル・レストラン・買い物など10場面)
- ✏️ 話した英語をその場でAIが添削(修正版・文法ポイント・ネイティブ表現・日本語解説)
- 🎤 音声で話して、音声で返ってくる(スマホ内蔵の音声認識・読み上げを使用 = 無料)
- 📒 会話終了ごとにふりかえりレビュー+覚えたいフレーズを復習ノートに自動保存
- 🆓 AI応答はGoogle Gemini APIの無料枠を使用。APIキーは端末内(localStorage)にのみ保存

## 使い方

1. スマホのブラウザでアプリのURLを開く
2. [Google AI Studio](https://aistudio.google.com/apikey) で無料APIキーを取得(要Googleアカウント)
3. アプリの初回ガイドに従ってキーを貼り付け
4. 「ホーム画面に追加」でアプリとして使える

## 開発メモ

ビルド不要の静的サイト(HTML/CSS/vanilla JS)。`?mock=1` を付けるとAPIなしのダミー応答でUIを確認できる。
