はじめの設定

会話AIを使うために、Googleの無料APIキーを1回だけ設定します。約2分で完了します。料金はかかりません。

  1. Google AI Studio↗ を開き、Googleアカウントでログイン
  2. 「APIキーを作成（Create API key）」をタップ
  3. 表示されたキー（AIza… で始まる文字列）をコピー
  4. 下の欄に貼り付けて保存

キーはこの端末の中だけに保存され、外部に送信されるのはGoogleのAIサーバーへの通信のみです。
